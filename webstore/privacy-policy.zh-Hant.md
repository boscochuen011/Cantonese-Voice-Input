# 私隱政策（範本）

最後更新日期：2026-02-12

本擴充功能「廣東話語音輸入」重視你的私隱。本政策說明資料如何被使用。

## 1. 我們收集甚麼
- 本擴充功能不會建立帳號系統，亦不會主動收集可識別個人身份資料。
- 擴充功能僅在本地儲存必要設定（例如語言設定、首次設定完成狀態）。

## 2. 語音資料處理
- 語音輸入功能透過瀏覽器 Speech Recognition API 提供。
- 我們不會把語音內容上傳到自家伺服器儲存或出售。
- 任何由瀏覽器服務提供者處理的資料，將受該服務提供者政策約束。

## 3. 權限用途
- `activeTab` / `scripting`：在目前頁面檢查輸入焦點並插入文字。
- `storage`：儲存本地設定。
- `sidePanel`：顯示操作面板。
- `http/https host permissions`：讓功能可在一般網站運作。

## 4. 第三方分享
- 我們不出售、出租或交易你的資料。
- 本擴充功能不內建第三方廣告追蹤 SDK。

## 5. 聯絡方式
- GitHub Issues：`https://github.com/boscochuen011/Cantonese-Voice-Input/issues`

> 使用前請先閱讀並確認你同意本政策。若你不同意，請停止使用本擴充功能。
