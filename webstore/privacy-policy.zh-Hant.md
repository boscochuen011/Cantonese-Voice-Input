# 私隱政策（範本）

最後更新：2026-02-12

本擴充功能「廣東話語音輸入」重視你的私隱。

## 1. 收集資料
- 本擴充功能不建立帳號，也不主動收集可識別身份資料。
- 只會在本地儲存必要設定（例如語言、首次設定狀態）。

## 2. 語音資料
- 語音輸入透過瀏覽器 Speech Recognition API 提供。
- 本擴充功能不會把語音內容上傳到自家伺服器保存或出售。

## 3. 權限用途
- `activeTab` / `scripting`：檢查焦點並插入文字。
- `storage`：儲存本地設定。
- `sidePanel`：顯示操作面板。
- `http/https host permissions`：讓功能於一般網站運作。

## 4. 第三方分享
- 不出售、不出租、不交易使用者資料。
