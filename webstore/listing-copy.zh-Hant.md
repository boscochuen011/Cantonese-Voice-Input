# Chrome Web Store 文案（繁體中文）

## 名稱
廣東話語音輸入

## 短描述（建議）
一鍵廣東話語音輸入，文字自動插入目前輸入欄；支援連按兩下 Control 快速啟動。

## 詳細描述（可直接貼上）
廣東話語音輸入是一個為日常打字而設的 Chrome 擴充功能，提供簡單直接的語音轉文字流程：

- 支援廣東話辨識（預設 `yue-Hant-HK`）
- 在網頁輸入欄中可直接插入文字
- 連按兩下 `Control` 即可快速啟動語音輸入
- 內建首次設定精靈，協助完成麥克風授權與使用教學

### 使用方式
1. 打開一般網站並點選文字輸入欄
2. 在側邊面板按「語音輸入」，或直接連按兩下 `Control`
3. 開始說話，停止後文字會自動插入到目前欄位

### 注意事項
- 瀏覽器內部頁（例如 `chrome://...`）不支援文字插入
- 主介面頁面或無輸入欄的位置，無法透過此輸入法直接插入
- 首次使用需授權麥克風
