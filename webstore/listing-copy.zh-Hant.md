# Store 文案（繁中）

## 名稱
廣東話語音輸入

## 短描述
一鍵廣東話語音輸入，並自動插入目前文字欄位；支援連按兩下 Control 快速啟動。

## 詳細描述
「廣東話語音輸入」是一個 Chrome 擴充功能，讓你在網頁輸入欄以語音快速打字。

- 支援廣東話辨識（預設 `yue-Hant-HK`）
- 側邊面板一鍵啟動語音輸入
- 在網頁內連按兩下 `Control` 可快速啟動
- 可直接插入 `input`、`textarea` 與 `contenteditable`
- 內建首次設定精靈（權限與使用提醒）

注意：
- `chrome://` 等瀏覽器內部頁面不支援插字
- 主介面或沒有文字輸入欄的位置不能直接插入文字
