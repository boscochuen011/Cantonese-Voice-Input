# 權限與資料使用說明（上架填寫用）

## 權限用途說明
- `activeTab`：
  僅在使用者當前分頁執行文字插入與焦點檢查。
- `scripting`：
  在當前頁面執行必要 script，用於辨識焦點欄位與插入文字。
- `storage`：
  儲存本地設定（例如首次設定完成狀態、語言設定）。
- `sidePanel`：
  顯示擴充功能的側邊操作面板。
- `host_permissions` (`http://*/*`, `https://*/*`)：
  只為了在一般網站提供語音輸入及插字功能，不用於追蹤或廣告。

## Data Disclosure 建議答案（請按你實際做法再次確認）
- 是否收集個人可識別資料：否（No）
- 是否將資料出售或轉移第三方：否（No）
- 是否用於廣告：否（No）
- 是否用於信用評分或借貸：否（No）

## 私隱聲明重點
- 語音辨識由瀏覽器 Speech Recognition API 執行。
- 擴充功能不會主動把語音內容儲存到你自己的伺服器。
- 只處理使用者當前輸入流程所需資料，並在本地儲存必要設定值。
